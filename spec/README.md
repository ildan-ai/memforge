# spec

The universal MemForge format. Anything in this folder is agent-neutral. If a concept only makes sense for one specific agent, it belongs under `adapters/<agent>/` instead.

Files:
- `SPEC.md` — current spec (v0.3.0).
- `taxonomy.yaml` — controlled vocabulary (topics, areas, priorities, synonyms, access labels, statuses, tiers).
- `VERSION` — current spec version.
